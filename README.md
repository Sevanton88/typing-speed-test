# TypeLab — Typing Speed Test

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A minimalist, editorial-style web application for measuring typing speed and accuracy. Built with vanilla JavaScript, featuring real-time statistics, multiple test modes, and a clean, distraction-free interface inspired by high-end editorial design.

![TypeLab Preview](https://via.placeholder.com/800x400/FAFAF8/1A1916?text=TypeLab+Typing+Speed+Test)

## ✨ Features

- **Real-Time Metrics**: Live WPM (words per minute), accuracy percentage, and error count during typing
- **Multiple Test Modes**:
  - Time-based: 15, 30, or 60 seconds
  - Word-based: 25 words
- **Personal Best Tracking**: Automatically saves and displays your highest WPM score
- **Test History**: Tracks the number of tests completed
- **Visual Feedback**: Color-coded letters (correct/incorrect), animated caret, and progress bar
- **Responsive Design**: Optimized for desktop and mobile devices
- **Smooth Animations**: Elegant transitions and micro-interactions
- **No Dependencies**: Pure HTML, CSS, and JavaScript — works offline

## 🎯 Usage

1. **Select a mode**: Choose from time-based (15s, 30s, 60s) or word-based (25 words) tests
2. **Start typing**: Click anywhere in the typing area or press any key to begin
3. **Monitor progress**: Watch real-time WPM, accuracy, and error metrics
4. **View results**: See detailed statistics and personal best achievements
5. **Try again**: Restart with the same mode or switch to a different one

### Controls

- **Backspace**: Correct mistakes

- **Mode buttons**: Switch between test types

## 📊 How It Works

TypeLab uses industry-standard typing test methodology:

- **WPM Calculation**: Based on characters per minute divided by 5 (standard word length)
- **Accuracy**: Percentage of correct keystrokes out of total keystrokes
- **Word Bank**: Curated list of common English words for realistic testing
- **Real-Time Updates**: Metrics update every second during timed tests
- **Persistence**: Uses localStorage for personal best and test count

# 🚀 TypeLab

TypeLab is a minimalist typing speed test built with vanilla HTML, CSS, and JavaScript. It lets users practice typing in timed or word-based modes while tracking speed, accuracy, errors, and personal best results.

---

## 🌍 Live Demo

Repository: https://github.com/Sevanton88/typing-speed-test

---

## ✨ Features

- Multiple test modes: 15 seconds, 30 seconds, 60 seconds, and 25 words
- Live typing stats including WPM, accuracy, and error count
- Result screen with final performance summary after each test
- Personal best and total test count saved with localStorage

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

---

## ⚡ Getting Started

1. Clone the repository  
2. Open `index.html` in your browser  
3. Click inside the typing area or press any key to start the test  

---

## 🎯 Project Goal

The goal of this project was to practice building an interactive frontend app with plain JavaScript, including DOM rendering, keyboard event handling, typing logic, live statistics, and browser storage for user progress.
